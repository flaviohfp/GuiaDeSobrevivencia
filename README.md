# GuiaDeSobrevivencia
Trabalho em grupo, da matéria de desenvolvimento web, do SENAI. Com os seguintes integrantes: Flavio, Maria, Yan, Davi.

Sobre o Projeto:

O Guia de Sobrevivência é uma página informativa sobre os mostros presentes em diversos jogos.

Nosso objetivo é criar um espaço onde o jogador possa consultar informações detalhadas sobre cada criatura, entendendo seus pontos fracos, táticas e forma de derrotar de cada criatura presente no site.

Caso o jogador precise enfrentar algum desses mostros, á página estrá disponível com orientações e estratégias que podem ajudá-lo a se preparar melhor. Assim, o foco é informar e auxiliar, oferecendo conhecimento para quem deseja explorar o mundo dos jogos com mais segurança e curiosidade.

Funcionalidades:

Exibição de mostro com as imagens de cada um e decrição.

Informações dos pontos fracos e táticas.

Dicas e estratégias cajo seja necessário enfrentá-lo.

Ter um layout fácil e prático de acessar e navegar.

Tecnologias Utilizadas:

HTML5 — estrutura da página

CSS3 — estilização e layout

JavaScript — interatividade
